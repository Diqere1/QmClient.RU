- 支持时间、观战数、地图进度、聊天、投票、输入覆盖、语音说话列表与媒体信息的预览和布局持久化

### 新增 - 视角与表现增强
- 相机漂移
- 动态 FOV
- 纵横比预设
- 果冻 Tee 形变
- 分身小窗

### 新增 - 其他
- 收藏地图管理与地图名一键复制能力
- 新增 `map_generator` 工具，可读取或生成基础 DDNet/Teeworlds 逻辑层地图骨架
- 新增纹理句柄 generation 机制，用于识别句柄复用后的过期状态，提升图形资源管理安全性
- 未发送聊天草稿保存

### 新增 - Tags 筛选功能
- **Configs 页面新增 Tags 筛选栏**
  - 支持按功能分类筛选配置项
  - 新增 10 个功能标签:
    - **Visual** - 视觉效果 (皮肤、彩虹、激光、Jelly Tee 等)
    - **HUD** - 界面显示 (计分板、灵动岛、玩家统计等)
    - **Input** - 输入优化 (Fast Input、Deepfly 等)
    - **Chat** - 聊天相关 (气泡、复读、关键词回复等)
    - **Audio** - 语音/音效 (RiVoice 语音聊天等)
    - **Automation** - 自动化功能 (Gores、自动切换等)
    - **Social** - 社交功能 (好友、饼菜单、屏蔽词等)
    - **Camera** - 相机/视野 (漂移、动态 FOV 等)
    - **Gameplay** - 游戏玩法 (武器轨迹、碰撞盒等)
    - **Misc** - 其他杂项

- **智能标签匹配系统**
  - 根据配置变量名自动推断所属分类
  - 支持通配符模式匹配 (如 `*laser*` 匹配所有激光相关配置)
  - 可通过 `CConfigTagsManager` 扩展更多标签规则

### 新增 - Qimeng 配置域 (QIMENG Domain)
- 创建独立的 `settings_qimeng.cfg` 配置文件
- 用于存储所有栖梦客户端 (QmClient) 特有的设置项
- 与 DDNet 原版和 TClient 配置完全分离

### 变更

  - **TClient**: TClient 特有设置项 (tc_ 前缀)

- 使用 `CONFIG_DOMAIN` 宏定义新的配置域
