# 设置页预热与 FBO 缓存实现计划

> **文档已过时** — 本文档内容不再反映当前代码状态，仅供参考。

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 实现 Settings-wide 预热与真实 FBO section 缓存，删除所有可见 compact/deferred 摘要文本优化。

**架构：** 先清理 Settings 页面中的摘要 fallback，再把 Settings section 注册、预热调度、FBO 缓存和 graphics render target 能力分层实现。预热覆盖整个 Settings 系统；FBO 以 section 为粒度缓存静态层；交互层继续走 DDNet immediate-mode UI。

**技术栈：** C++、DDNet immediate-mode UI、CMake、GoogleTest、OpenGL backend、Vulkan backend、QmClient Windows 构建脚本。

```cpp
#ifndef GAME_CLIENT_COMPONENTS_SETTINGS_WARMUP_H
#define GAME_CLIENT_COMPONENTS_SETTINGS_WARMUP_H

#include <functional>
#include <vector>

enum class EClassicSettingsPage
{
	GENERAL,
	PLAYER,
	TEE,
	APPEARANCE,
	GRAPHICS,
	SOUND,
	DDNET,
	CUSTOM,
	CONTROLS,
	TCLIENT,
	QMCLIENT,
	TCLIENT_PROFILES,
};

struct SSettingsWarmupSection
{
	EClassicSettingsPage m_Page;
	const char *m_pName;
	int m_Priority;
	std::function<double()> m_WarmupFn;
	bool m_bDone = false;
};

class CSettingsWarmupScheduler
{
public:
	void RegisterSection(SSettingsWarmupSection Section);
	void SetLastSessionPage(EClassicSettingsPage Page);
	bool WarmupFrame(double BudgetMs);
	void Reset();

private:
	std::vector<SSettingsWarmupSection> m_vSections;
	EClassicSettingsPage m_LastSessionPage = EClassicSettingsPage::GENERAL;
};

#endif
```

- [ ] **步骤 4：实现最小调度器逻辑**

创建 `src/game/client/components/settings_warmup.cpp`：

```cpp
#include "settings_warmup.h"

#include <algorithm>

void CSettingsWarmupScheduler::RegisterSection(SSettingsWarmupSection Section)
{
	m_vSections.push_back(std::move(Section));
}

void CSettingsWarmupScheduler::SetLastSessionPage(EClassicSettingsPage Page)
{
	m_LastSessionPage = Page;
}

bool CSettingsWarmupScheduler::WarmupFrame(double BudgetMs)
{
	std::stable_sort(m_vSections.begin(), m_vSections.end(), [&](const SSettingsWarmupSection &Left, const SSettingsWarmupSection &Right) {
		const bool LeftLast = Left.m_Page == m_LastSessionPage;
		const bool RightLast = Right.m_Page == m_LastSessionPage;
		if(LeftLast != RightLast)
			return LeftLast;
		return Left.m_Priority < Right.m_Priority;
	});

	double UsedMs = 0.0;
	for(auto &Section : m_vSections)
	{
		if(Section.m_bDone)
			continue;
		if(UsedMs >= BudgetMs)
			return false;
		const double CostMs = Section.m_WarmupFn ? Section.m_WarmupFn() : 0.0;
		Section.m_bDone = true;
		UsedMs += CostMs;
	}
	return true;
}

void CSettingsWarmupScheduler::Reset()
{
	for(auto &Section : m_vSections)
		Section.m_bDone = false;
}
```
